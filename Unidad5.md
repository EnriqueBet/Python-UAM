## Funciones 

Usar funciones nos permiten dividir el código en bloques que podemos reutilizar nuestro código. Las funciones se declaran como: 

~~~py
def f():
  print("Hola funcion")
  
 def suma(a,b):
  return a+b
 
# Calling a function

fun1 = f()
fun2 = suma(2,3)

print(fun1)
print(fun2)
print(suma(34,234))
~~~
