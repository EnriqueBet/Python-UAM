## Funciones 

Usar funciones nos permiten dividir el código en bloques que podemos reutilizar nuestro código. Las funciones se declaran como: 

~~~py
def f():
  print("Hola funcion")
  
 def suma(a,b):
  return a+b
 
 def imprimir(texto, veces):
  print(veces * texto)
 
# Calling a function

fun1 = f()
fun2 = suma(2,3)
fun3 = imprimir("hola", 4)

print(fun1)
print(fun2)
print(suma(34,234))
print(fun3)
~~~
